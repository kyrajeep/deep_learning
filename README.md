# Deep Learning Repository

A comprehensive collection of deep learning projects, tutorials, and exercises using **PyTorch** and **TensorFlow**. This repository contains implementations of neural networks, computer vision models, natural language processing tasks, and various machine learning concepts.

## Contents

### Root Level Scripts
- **`chatbot.py`** - Chatbot implementation
- **`gradient_descent.py`** - Gradient descent algorithm implementation
- **`linear_svm.py`** - Linear Support Vector Machine
- **`multi.py`** - Multi-class classification
- **`cython_tutorial.ipynb`** - Tutorial on Cython optimization

### Learning Notebooks
- **`PyTorch1.ipynb`** - Introduction to PyTorch
- **`Copy_of_Intro_to_Neural_Nets.ipynb`** - Neural network fundamentals
- **`Copy_of_Binary_Classification.ipynb`** - Binary classification techniques
- **`Copy_of_Course_3_Week_1_Lesson_1.ipynb`** - Course material
- **`Diabetes.ipynb`** - Diabetes prediction project
- **`hyperparameter_tuning_tensorflow.ipynb`** - TensorFlow hyperparameter optimization

### NLP & Sentiment Analysis
- **`netflix_reviews.ipynb`** - Sentiment analysis on Netflix reviews
- **`netflix_common_words.ipynb`** - Analysis of common words in Netflix reviews

### Deep Learning V2 PyTorch (Advanced Projects)
The `deep-learning-v2-pytorch/` directory contains comprehensive project implementations:

- **`intro-to-pytorch/`** - Introduction to PyTorch fundamentals
  - Tensors, neural networks, training, validation, and model saving
  
- **`convolutional-neural-networks/`** - CNN implementations
  - MNIST MLP, CIFAR CNN, and convolution visualization
  
- **`recurrent-neural-networks/`** - RNN tutorials and projects
  
- **`sentiment-rnn/`** - Sentiment analysis using recurrent networks
  
- **`sentiment-analysis-network/`** - Sentiment classification networks
  
- **`attention/`** - Attention mechanisms and transformers
  
- **`transfer-learning/`** - Transfer learning techniques
  
- **`cycle-gan/`** - CycleGAN implementation (image-to-image translation)
  
- **`dcgan-svhn/`** - DCGAN for SVHN dataset
  
- **`gan-mnist/`** - Generative Adversarial Networks on MNIST
  
- **`project-bikesharing/`** - Time series prediction project
  
- **`project-dog-classification/`** - Dog breed classification
  
- **`project-face-generation/`** - Face generation using GANs
  
- **`project-tv-script-generation/`** - Text generation for TV scripts
  
- **`autoencoder/`** - Autoencoders (linear, convolutional, denoising)
  
- **`batch-norm/`** - Batch normalization techniques
  
- **`weight-initialization/`** - Neural network weight initialization
  
- **`word2vec-embeddings/`** - Word embeddings and representation learning
  
- **`keras/`** - Keras-based implementations
  
- **`tensorflow/`** - TensorFlow implementations

## Technologies
- **PyTorch** - Deep learning framework
- **TensorFlow/Keras** - Alternative deep learning framework
- **Python** - Programming language

## Getting Started
Clone the repository and explore the notebooks and scripts. Each project contains its own documentation and can typically be run independently.

## Requirements
- Python 3.x
- PyTorch
- TensorFlow
- NumPy, Pandas, Matplotlib
- Jupyter Notebook

See `deep-learning-v2-pytorch/requirements.txt` for detailed dependencies.
